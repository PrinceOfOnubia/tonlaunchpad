# Tonic

One sip. Up only.

Tonic is a TON-native bonding-curve meme launchpad MVP built with Next.js, TypeScript, and TailwindCSS.

## Run locally

```bash
npm install
npm run dev
```

## MVP scope

- Landing page
- Create token page
- Token detail page
- Bonding curve buy/sell UI
- Trending and latest launch sections

Smart contract and TON Connect integrations are represented by clean placeholder services and components for future implementation.
